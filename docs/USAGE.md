## Getting Started

### First Launch
When you first open FinanceBuddy, you'll see:
1. **Welcome Screen**: Introduction to your financial wellness journey
2. **Three Main Tabs**: Chat, Expenses, and Plant Care
3. **Your Plant**: Starting as a tiny seed, ready to grow with your financial wellness

### Initial Setup
1. **Grant Permissions**: Allow microphone access for voice features (optional)
2. **Set Currency**: Default is South African Rand (ZAR)
3. **Create First Expense**: Log your first expense to start earning points
4. **Meet Your Plant**: Tap on your plant to see detailed information

### ?? Expense Tracking

Comprehensive expense management with gamification integration.

#### Adding Expenses

**Manual Entry**
1. **Navigate to Expenses Tab**
2. **Tap "Add Expense" Button**
3. **Fill in Details**:
   - Amount (in your default currency)
   - Category (Food, Transport, Entertainment, etc.)
   - Date (defaults to today)
   - Optional note for context
4. **Save**: Automatically earns 5+ points

**Voice Entry** (if microphone permission granted)
1. **Tap Microphone Icon** on expense entry screen
2. **Speak Expense**: "Fifty rand for lunch at the cafe"
3. **Review and Confirm**: App parses amount, category, and note
4. **Save**: Faster entry with same point rewards

## Troubleshooting

### ?? Common Issues and Solutions

#### App Performance

**Problem**: App running slowly
**Solutions**:
- Close other running apps to free memory
- Restart the FinanceBuddy app
- Check for app updates
- Restart your device if issues persist

**Problem**: Plant not updating after earning points
**Solutions**:
- Tap the refresh button in Plant Care tab
- Navigate away and back to the tab
- Check internet connection for data sync
- Use "Refresh Plant" button in testing section

#### Data Synchronization

**Problem**: Expenses not syncing between devices
**Solutions**:
- Check internet connectivity
- Ensure you're logged into the same account
- Manually trigger sync in settings
- Contact support if issues persist

**Problem**: Points not reflecting in plant growth
**Solutions**:
- Use "Refresh Plant" function in Plant Care tab
- Check point calculation in detailed stats
- Verify activities are being tracked properly
- Reset and recalculate if necessary

#### AI Chat Issues

**Problem**: Money Mentor not responding
**Solutions**:
- Check internet connection
- Verify API service status
- Try rephrasing your question
- Check for service outages in settings

**Problem**: Voice input not working
**Solutions**:
- Check microphone permissions
- Ensure microphone is not being used by other apps
- Test device microphone in other apps
- Use text input as alternative

#### Plant and Gamification

**Problem**: Points not being awarded
**Solutions**:
- Verify activity completion (expense saved, chat sent, etc.)
- Check point calculation in Plant Care testing
- Review activity history in plant details
- Reset gamification profile if data is corrupted

**Problem**: Plant stage not advancing
**Solutions**:
- Check point thresholds for next stage
- Use "Check for Level Up" in Plant Care tab
- Review point calculation accuracy
- Manually refresh plant status
